Proyecto de analisis de datos

Descripción:Es un proyecto de análisis de datos en el que procesé, limpié y modelé un conjunto de datos sobre insuficiencia cardíaca.
Este conjunto de datos incluye registros médicos de 299 pacientes y 13 características clínicas. Utilicé la librería datasets de Huggingface para descargar el conjunto de datos y lo almacené en una estructura similar a un diccionario. Convertí la lista de edades a un arreglo de NumPy y calculé el promedio de edad de las personas participantes en el estudio. Este proyecto me permitió aplicar los conceptos de análisis de datos y machine learning.
